##  Project Overview

This project is developed as part of **CODTECH Internship - Task 1**, where the objective is to build a responsive webpage that fetches and displays data from a public API.

The application is a **Weather Web App** that retrieves real-time weather data from a public API and displays it dynamically on the webpage.

---

##  Objective

To develop a responsive web application that:

* Fetches data from a public API
* Displays the data dynamically
* Provides a smooth and interactive user experience

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API:** Weather API (e.g., OpenWeatherMap)
* **Tools:** VS Code, Browser

---



##  How It Works

1. User enters a city name
2. Application sends a request to the Weather API
3. API returns weather data in JSON format
4. Data is processed and displayed on the webpage
